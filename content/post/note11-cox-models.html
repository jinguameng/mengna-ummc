Cox Models = Propotional Hazards Models

Modeling the Hazard Function1

  • Survival analysis methods, such as proportional hazards regression differ from logistic regression by assessing a rate instead of a proportion.

  • Proportional hazards regression, also called Cox regression, models the incidence or hazard rate, the number of new cases of disease per population at-risk per unit time. If the outcome is death, this is the mortality rate.

  • The hazard function is the probability that if a person survives to t, they will experience the event in the next instant.

\(\lambda (t|X_{1i}, X_{2i}, ..., X_{ki}) = \lambda _0 (t)exp(\beta_1X_{1i} + \beta_2X_{2i} + ... +\beta_kX_{ki})\)

\(log(\frac{\lambda (t|X_{1i}, X_{2i}, ..., X_{ki})}{\lambda _0 (t)}) = exp(\beta_1X_{1i} + \beta_2X_{2i} + ... +\beta_kX_{ki})\)

  • \(\lambda (t|X_{1i}, X_{2i}, ..., X_{ki}):\) this is the the hazard function for the ith person at time t,i=1,2,⋯,n.

  • \(\lambda _0 (t):\) this is the baseline hazard function at time t, which is analogous to the intercept term in a multiple regression or logistic regression model. And it is unknown.

  • \(log(\frac{\lambda (t|X_{1i}, X_{2i}, ..., X_{ki})}{\lambda _0 (t)}):\) this is the log of hazard ratio. The ratio of hazard functions can be considered a ratio of risk functions.

  • There is no particular probability model selected to represent the survival times.

  • But there is a big assumption: the hazard for any individual is a fixed proportion of the hazard for any other individual. (i.e., proportional hazards). Notice if \(λ_0(t)\) is the hazard function for a subject with all the predictor values equal to zero and \(λ_1(t)\) is the hazard function for a subject with other values for the predictor variables, then the hazard ratio depends only on the predictor variables and not on time t. This assumption means if a covariate doubles the risk of the event on day one, it also doubles the risk of the event on any other day.

Output Interpretation

  • Hazard Ratio: similar with odds ratio.

  • Continuous: A unit increase in X is associated with a YY% increase/decrease in the hazard of (event) on average.

  • Binary: On average, A has YY% higher/lower hazard of (event) compared to B.
